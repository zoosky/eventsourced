org.eligosource.eventsourced.core

Context

case class Context(journal: ActorRef, processors: Map[Int, ActorRef], channels: Map[String, ActorRef], producer: ActorRef, system: ActorSystem) extends Product with Serializable

A context for Eventsourced processors and Channels that are used by processors to emit event Messages to destinations. A Destination can be any actor including a processor of this context.

journal

Journal for this context. A journal must not be shared between contexts.

processors

Actors modified with the Eventsourced trait.

channels

Channel actor map used by processors to emit event messages. Keys of the channels map are application- defined channel names.
